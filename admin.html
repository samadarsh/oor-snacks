First time setup

  1. Supabase Dashboard → Authentication → Users
  2. Add user → Create new user
  3. Enter email + password
  4. Turn on Auto Confirm User (required)
  5. Sign in here with that exact email and password

Still failing? In Users, open the user → Send password recovery or delete and recreate with Auto Confirm on.
